# 示范案例：方法论操作演示

> **重要说明：** 本文件展示如何将分析框架应用于实际操作，目的是让使用者理解方法，而非提供真实的人物心理数据。
>
> 下方内容是**格式示范**，使用假设性场景演示每个步骤的操作逻辑。若用于真实人物分析，请重新收集该人物的实际语料并重新编码。

---

## 示范：如何执行 Step 0（先验申报）

**场景：** 对一位科技公司创始人进行分析

```
[先验申报]
目标人物：[姓名]
既有印象：
1. 公开形象以技术理想主义为主
2. 倾向于长期主义叙事
3. 在媒体上偏向低调，较少主动发声
4. 对监管议题持复杂立场
5. 重视安全与责任的表达
[/先验申报]
```

**关键原则：** 先验申报越诚实越好。列出真实的预设，分析完成后才能知道哪些被推翻。

## 示范：如何记录矛盾日志

**矛盾点记录格式：**

```
主题：对监管介入的态度

场合 A（2021年，面向开发者社区）：
「我们需要空间去实验，过早的监管会扼杀创新的可能性。」

场合 B（2024年，面向国会听证）：
「我们主动呼吁针对我们这个行业的强制性监管框架。」

可能解释：
- 情境策略：两个受众期待不同，表达相应调整（最可能）
- 真实转变：2021-2024 之间观点发生了实质性演化（需要中间年份语料验证）
- 自我矛盾：认知层面存在不一致（证据不足以支持）

支持「情境策略」的证据：在同期（2024年）面向开发者的访谈中，仍使用「灵活监管」等表达——说明核心立场可能未变，只是框架在调整。
```

## 示范：如何从引证到结论（Layer 1 局部）

**目标维度：** 时间尺度

**原始引证 1**（A级来源，2023年）：
> 「我在想的不是今年的路线图，而是十年后这项技术在社会中扮演什么角色。」

**原始引证 2**（A级来源，2024年）：
> 「短期的市场份额对我来说不是核心指标，我更关心五到十年后这个行业的格局。」

**原始引证 3**（B级来源，2022年）：
> 「如果一个决策在三个月内看起来好，但十年内是错的，我们不会做。」

**编码结论：**
在现有语料中，该人物在谈到目标和决策时，一致使用5-10年以上的时间框架。三条引证来自不同场合（播客、媒体深访、内部对谈），时间跨度3年，置信度：**高**。

**标注：** `十年+视角`，置信度：高，引证数：3

## 示范：如何写行为预测

**错误示范（模糊）：**
> 此人可能会在压力下做出稳健的决策。

**正确示范（具体可检验）：**
> 在面临短期业绩压力与长期战略投资的取舍时，此人倾向于优先保护长期布局，即使这意味着接受短期指标的下滑——因为其决策框架以十年+时间尺度为基准。可预期的行为：拒绝为了季度数据而削减研发投入；在股东压力下维持长期战略不变，并用长期叙事来解释短期表现。
