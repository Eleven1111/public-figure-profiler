# Hermann 领导特质分析（Leadership Trait Analysis, LTA）

> 来源：Margaret G. Hermann (1980, 2003), Profile of the World's Political Leaders.
> 本框架是**政治心理学最成熟的领导人侧写方法**，基于对演讲和访谈的内容分析。
> 在公开人物分析中广泛用于预测决策行为。

## 七大特质维度

LTA 通过**语言内容分析**测量 7 个特质，每个特质都有明确的操作化标记词。

### 1. 权力需求（Need for Power, nPow）

**定义：** 控制他人、影响事件、在权力结构中占优势的欲望。

**高倾向标记：**
- 主动权力动词：「控制」「主导」「影响」「改变」「说服」「决定」
- 胜负导向：「赢得」「战胜」「压制」
- 对权力游戏的精细描述（谈判手段、联盟策略）

**低倾向标记：**
- 协商导向、共识导向
- 很少用主动权力动词
- 对权力表达谨慎甚至回避

**标注：** `高权力需求` / `中等` / `低权力需求`

### 2. 控制信念（Belief in Ability to Control Events, BACE）

**定义：** 相信自己能控制事件走向的程度。与内控相似但专指**国家/组织层面**的宏大事件。

**高信念标记：**
- 第一人称主动语态：「我将」「我们会」「我能」
- 计划性表达：「我的战略是」「接下来我要」
- 对未来的确定性表达

**低信念标记：**
- 被动语态 / 宿命感
- 「不得不」「形势所迫」
- 更多描述环境制约

**标注：** `高控制感` / `中等` / `低控制感`

### 3. 概念复杂度（Conceptual Complexity, CC）

**定义：** 能容纳多少种并存的观点、多维度分析问题的能力。

**高复杂度标记：**
- 使用高层限定词：「可能」「某种程度上」「在某种条件下」「通常但不总是」
- 承认多方面影响：「一方面…另一方面…」
- 对概念的多种定义和情境区分

**低复杂度标记：**
- 使用绝对化限定词：「绝对」「总是」「永远」「毫无疑问」
- 二元对立思维：正确/错误，朋友/敌人
- 单一因果解释

**标注：** `高复杂度` / `中等` / `低复杂度（教条化）`

### 4. 自信心（Self-Confidence, SC）

**定义：** 对自己能力和判断的确信程度。

**高自信标记：**
- 声明式表达（Declarative，非 hedging）
- 主体代词高频：「我」「我的」（相对语料总量）
- 不寻求他人确认

**低自信标记：**
- 多用限定词、商量语气
- 频繁引用权威支持自己观点
- 主体代词密度低

**标注：** `高自信` / `中等` / `低自信 / 自我怀疑`

### 5. 任务聚焦 vs 群体聚焦（Task/Affective Focus, TASK）

**定义：** 关注任务完成还是关注人际关系和团体氛围。

**任务聚焦标记：**
- 目标、指标、绩效、完成、效率等词
- 对人际问题以"解决"框架描述
- 很少谈团队感受

**群体聚焦标记：**
- 频繁提"我们""团队""家人"
- 关注成员感受、氛围、归属感
- 表达情感的比重高

**标注：** `任务聚焦` / `平衡` / `群体聚焦`

### 6. 对他者的不信任（Distrust of Others, DIS）

**定义：** 倾向于怀疑他人动机、假设他人有敌意。

**高不信任标记：**
- 频繁提「阴谋」「背后有人」「真实意图是」
- 对批评者进行动机归因（而非回应论点）
- 敌友二分叙事

**低不信任标记：**
- 默认善意推定
- 很少进行动机猜测
- 愿意相信对方所说的理由

**标注：** `高不信任（多疑）` / `中等` / `低不信任（善意推定）`

### 7. 内部群体偏好（In-Group Bias, IGB）

**定义：** 将自己所属群体理想化、与外群体对立的倾向。

**高偏好标记：**
- "我们"优越性表达：「我们的文化/公司/国家独特」
- 对外群体贬损或刻板化
- 历史宏大叙事（"自古以来"）

**低偏好标记：**
- 跨群体包容表达
- 对自己群体的缺陷有批评
- 寻找共同基础

**标注：** `高内群偏好` / `中等` / `低内群偏好（跨群体包容）`

## 两种领导类型（组合解读）

基于 7 特质的组合，Hermann 提出两种典型领导类型：

### A 型：挑战约束型（Challenge Constraints）
**特征组合：** 高 BACE + 高 nPow + 低 DIS
**行为预测：** 主动突破限制，推动变革，有风险偏好

### B 型：尊重约束型（Respect Constraints）
**特征组合：** 低 BACE + 中 nPow + 高 CC
**行为预测：** 在既有框架内运作，渐进式变革

这两种组合是**起点**，不是刻板标签，真实领导人常为混合型。

## 最小引证门槛

- 每个特质：≥3 条引证
- 概念复杂度（CC）需要覆盖 ≥2 个不同话题域（避免因领域熟悉度导致误判）
