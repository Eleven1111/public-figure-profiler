# 核心编码框架（Core Codebook）

> 行为编码 + 认知结构 + 心理表征 + 受众敏感度。适用于任意有足够公开语料的人物。
> 每条结论必须附原文引证（≥10字）并标注 `[Snn]` 来源编号。

---

## 置信度定义（全手册通用）

| 置信度 | 标准 |
|--------|------|
| **高** | ≥3 个跨情境一致引证（不同场合、不同受众均出现） |
| **中** | 1-2 个引证，或多个引证但情境单一 |
| **低** | 推断，无直接引证，必须标注推断路径 |

**引证定义：** 直接引用原文片段（≥10 个字），标注 `[Snn, 年份, 场合]`。摘要性描述、行为观察不算独立引证。

**跨情境定义：** 来源场合不同（播客 vs 听证会 vs 深访算跨情境；同一播客的两集不算）。受众类型不同可以加分。

## 语料不足处理规则

| 语料状态 | A/B级来源数 | 处理方式 |
|---------|-----------|---------|
| **充分** | ≥3篇，时间跨度≥2年 | 正常执行全部维度分析 |
| **偏少** | 1-2篇A/B级 | 整体置信度上限降为「中」；在报告开头标注「语料偏少」 |
| **不足** | 仅C/D级来源 | 所有维度最高置信度为「低」；标注「探索性草稿模式」 |

对于语料不足的维度（该维度无有效引证）：标注为 `N/A — 无有效引证`，不做强行推断。

## Layer 1：人格与行事风格

### 1.1 能量方向

**定义：** 主动发起互动 vs 倾向回应；在公开表达中的存在感与主动性。

**判断依据：**
- 主动发起：率先提出话题、主动打断、话题掌控感强
- 回应为主：等待提问、较少主动引申

**标注：** `内向倾向` / `外向倾向` / `情境依赖`

### 1.2 风险偏好

**定义：** 在面对不确定性和下行风险时，语言表达中呈现的态度。

**判断依据：**
- 求稳：频繁使用「稳健」「渐进」「可控」「预防」等词；强调下行保护
- 中性：对风险和机会的语言权重大致对等
- 求险：强调「突破」「速度」「先发优势」「容错」；轻描下行风险

**标注：** `求稳` / `中性` / `求险`

### 1.3 时间尺度

**定义：** 规划和决策时习惯使用的时间视野。

**判断依据：**
- 季度视角：讨论当前指标、短期执行
- 年度视角：讨论产品路线图、组织调整
- 十年+视角：讨论文明走向、技术代际影响

**标注：** `季度视角` / `年度视角` / `十年+视角`

### 1.4 决策模式

**定义：** 如何描述自己或团队做重大决策的过程。

**判断依据：**
- 独断：「我决定」「我认为应该」，强调个人判断
- 集体：「我们讨论」「团队共识」，强调共同决策
- 数据驱动：「数据表明」「实验结果显示」，强调客观依据
- 实验试错：「我们尝试」「先做再看」「快速迭代」

**标注：** `独断` / `集体` / `数据驱动` / `实验试错` / `混合`

## Layer 2：认知结构

### 2.1 主用世界模型

**定义：** 此人习惯用哪种底层逻辑来理解复杂问题。

| 模型 | 核心逻辑 | 典型用词 |
|------|---------|---------|
| 技术–经济模型 | 规律、规模、资源投入、收益递增/递减 | scaling、compute、return、efficiency |
| 制度–博弈模型 | 规则、利益方、谈判、国家竞争 | regulation、incentives、race、governance |
| 道德–价值模型 | 红线、责任、权利、代际公平 | safety、harm、obligation、future generations |

同一人可混用多种模型，但通常有主导倾向。记录每种模型出现的频率和语境。

### 2.2 不确定性处理

- **期望值思维：** 把决策视为概率×收益的计算，接受不确定性，追求期望最优
  - 典型表达：「从概率上来说」「大概率是」「值得冒这个险因为…」

- **红线约束思维：** 某些结果无论概率多低都不可接受，设定绝对边界
  - 典型表达：「无论如何我们不会」「这是底线」「不值得以……为代价」

### 2.3 因果推理密度

- 高密度：频繁使用「因为…所以」「如果…那么」「导致」「由此推断」等词，推理链长
- 中等：因果连词出现，但推理链短（1-2步）；或仅在特定话题密集
- 低密度：更多描述性表达，少用因果连词

## Layer 3：心理表征与叙事

### 3.1 核心隐喻

**操作：** 收集所有明确的比喻和类比，记录原文。
- 隐喻揭示心理表征：把 AI 比作「核武器」vs「基础设施」vs「孩子」，反映截然不同的责任感与控制欲
- 关注重复出现的隐喻（≥3次），这是稳定的心理表征

### 3.2 自我角色定位

**常见角色类型：**
- **守门人：** 我负责防止坏事发生，设定边界
- **建设者：** 我负责创造，推动进步
- **协调者：** 我负责在多方之间建立共识
- **研究者：** 我负责理解真相，追求知识
- **反对者：** 我站在某种力量的对立面

### 3.3 核心价值词频

1. 统计高频词（相对频率，而非绝对频率）
2. 分析每个价值词的典型语境
3. 推断价值排序：当两个价值词同时出现时，哪个更优先？

## 受众敏感度维度

| 等级 | 描述 | 诊断意义 |
|------|------|---------|
| **高敏感** | 核心观点不变，但框架、词汇、举例随受众调整 | 战略型沟通者，高度情境意识 |
| **中等** | 有轻微调整，主要风格基本一致 | 大多数公开人物的典型状态 |
| **低敏感** | 跨受众表达高度一致 | 原则驱动，或缺乏情境感知 |
| **异常高敏感** | 核心立场本身随受众变化 | 机会主义，或极强的自我保护动机 |

**判断方法：** 比较同一主题在：技术受众 vs 监管者 vs 大众媒体 vs 投资者 的表达。关注核心词汇、论点框架、核心立场是否变化。

## 矛盾点分析（跨维度 · 最高信号密度 · ACH 矩阵）

**触发条件：** 同一人物对同一问题在不同场合说法不一致，或言行背离。

**分析步骤（竞争性假设分析，Heuer ACH）：**
1. 记录两种表述/行为的原文、场合、受众、`[Snn]`
2. 列出候选假设（≥4）：H1 情境策略 / H2 真实认知转变 / H3 自我矛盾 / H4 利益驱动
3. 全部相关证据逐条对每个假设标 `+`（一致）/ `−`（不一致）/ `0`（无关），形成矩阵
4. 选**被否证最少**的假设（− 最少），不是佐证最多的
5. 输出矩阵 + 次优假设 + 判别所需的缺失证据

---

## 时间轴分析（Deep Mode）

1. 将语料按年份分组
2. 在同一维度上，比较不同时期的表达
3. 若发现显著变化，记录：变化内容 + 时间节点 + 可能触发事件
4. 区分：观点演化（渐进）vs 突变（事件触发）

---

## 最小引证门槛

- 每个 Layer 子维度（1.1/1.2/.../3.3）：≥2 条引证
- 受众敏感度：需至少覆盖 2 种不同受众类型
- 矛盾点：必须同时有两侧原文引证
