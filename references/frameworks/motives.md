# 三动机分析（Motive Imagery Analysis）

> 来源：David McClelland (1961, 1985), *Human Motivation*; Winter (1991) 远距离动机测量法（Motive Imagery Coding）。
> LTA 只测权力需求（nPow）一项，本框架补全成就（nAch）与亲和（nAff），
> 回答侧写的根本问题：**驱动此人的到底是成就感、控制感，还是归属感？**

## 三大动机维度

每个动机通过语料中**意象（imagery）出现频率**测量——不是此人说自己要什么，
而是他在自由叙述时**不自觉反复回到的主题**。

### 1. 成就动机（Need for Achievement, nAch）

**定义：** 追求卓越标准、把事情做得更好的内在驱动。参照系是**任务本身**，不是他人。

**高倾向意象：**
- 与卓越标准比较：「做到极致」「世界级」「比上一版好」「最优解」
- 对效率/方法改进的自发兴趣：「更聪明的做法」「重新发明」
- 独特成就叙事：强调"从未有人做到"，而非"打败了谁"
- 对失败的任务化归因：复盘方法，而非追究人

**低倾向意象：** 很少谈"做得更好"，目标以维持/合规为主。

**标注：** `高` / `中` / `低`

### 2. 权力动机（Need for Power, nPow）

**定义：** 影响、控制他人和事件的驱动。参照系是**他人的反应**。

**高倾向意象：**
- 强力动作：「控制」「主导」「征服」「碾压」「说服了他」
- 对他人情绪的影响叙事：「震惊了所有人」「让他们意识到」
- 声望意象：与大人物的关系、排场、头衔的自发提及
- 主动给予未被请求的帮助/指导（权力的温和形态）

**标注：** `高` / `中` / `低`（与 lta.md 的 nPow 交叉校验，不一致时说明）

### 3. 亲和动机（Need for Affiliation, nAff）

**定义：** 建立、维持温暖关系的驱动。参照系是**关系本身**。

**高倾向意象：**
- 对友谊/陪伴的自发提及：「老朋友」「一起走过」「家人般的团队」
- 分离焦虑叙事：对离开、散伙、背叛的情绪化描述
- 决策中的关系权重：「不想让他失望」「为了团队」

**低倾向意象：** 关系以工具性框架描述（「有用的人脉」），少有情感语言。

**标注：** `高` / `中` / `低`

---

## 动机组合解读（核心交付）

单个动机分数意义有限，**组合模式**才是预测器：

| 组合 | 典型行为模式 |
|------|-------------|
| 高 nAch + 低 nPow | 工匠/创始人型：做事优先，对管理与政治不耐烦，倾向亲自下场 |
| 高 nPow + 高自我克制 | 制度建设者：通过组织和规则放大影响力（Winter 称"社会化权力"） |
| 高 nPow + 低自我克制 | 个人化权力：冲动扩张、个人忠诚网络、对制衡的敌意 |
| 高 nAch + 高 nPow | 帝国建造者：既要做成事又要控制局面，授权困难，接班人问题高发 |
| 低 nAff + 高 nPow | 孤立决策者：危机中不咨询，关系破裂成本低，转向激烈 |
| 高 nAff + 高 nPow | 联盟型领导：靠忠诚网络运作，对"背叛"反应远超利益受损本身 |

**判断"利益取向"的钥匙：** 此人在动机满足与物质利益冲突时选了什么？
（如：为控制权放弃套现机会 = nPow > 财务利益；为做成产品稀释股权 = nAch > nPow）

## 最小引证门槛

- 每个动机维度：≥3 条引证，且至少 2 个不同场合
- 组合解读：必须引用至少 1 个**动机间冲突的实例**（两个动机不可兼得时的实际选择）
- 不达门槛标 `N/A — 语料不足`
