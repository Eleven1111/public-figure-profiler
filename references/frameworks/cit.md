# Flanagan 关键事件技术（Critical Incident Technique, CIT）

> 来源：John C. Flanagan (1954), "The Critical Incident Technique", Psychological Bulletin.
> 原用于航空事故分析，后扩展为行为取向的结构化分析方法。
> **核心原则：真实行为 > 自我陈述。** 人说自己怎样可能造假，但"他当时做了什么"无法修改。

## 什么是"关键事件"？

**关键事件（Critical Incident）** 必须满足：

1. **可观察的具体事件**：有明确时间、地点、在场人物、行为
2. **有意义的结果**：事件导致了明显的后果（成功/失败/转折）
3. **揭示个人特征**：事件中行为能反映当事人的能力/价值/性格

**反例**：「他经常在会议上说话」← 模糊，非关键事件
**正例**：「2019 年 5 月孟晚舟被捕次日，任正非在全员信中说『让不适应的员工离开』」← 具体、有后果、揭示价值观

## 四类最高信号事件

### 第一类：危机/失败事件

公开人物在**失败、被指控、被批评、产品重大事故**时的**首次公开反应**。
- 为什么高信号：排演时间短、情绪真实度高、防御机制来不及包装
- 关键观察点：是否道歉？归因方向？是否攻击对方？

**示例：** 扎克伯格在 Cambridge Analytica 事件后的国会听证——观察其面对攻击性提问时的情绪调节模式。

### 第二类：意外揭露事件

- 被媒体**突然披露**其未公开的言论（录音、内部邮件、聊天截图）
- **泄露内部讲话**（非演讲稿，是真实对话）
- 无准备场合的**即兴回应**

这类事件的价值：绕过了"公关包装层"，能看到真实的话语风格和价值排序。

### 第三类：决定性选择点

- 在两难情境中**公开做出的选择**（且当事人清楚代价）
- 例如：公司内部是否裁员、是否配合监管、是否发布某产品

关注：
- 选择背后的排序（他把什么放第一位？）
- 事后是否后悔？是否合理化？

### 第四类：长期承诺兑现/违背

- 曾公开承诺过的事，后来如何处理？（兑现 / 违背 / 默默撤回 / 强行合理化）
- 这是最难伪造的信号：**时间跨度本身过滤掉了表演**

## CIT 执行流程

### Step 1：识别候选事件

从语料中标记所有**符合四类定义**的事件片段。不是所有发生在人物身上的事都是关键事件。

### Step 2：事件编码（STAR 框架扩展）

对每个关键事件，记录：

| 字段 | 内容 |
|------|------|
| **Situation** | 事件背景、触发条件 |
| **Task/Tension** | 当事人面临的选择或挑战 |
| **Action** | **实际采取的行为**（原文描述，不转述） |
| **Result** | 事件的后果 |
| **Reflection** | 当事人事后如何解释/合理化（同 Action 比较） |
| **信号提取** | 这个事件揭示了什么？映射到哪个维度？ |
| **来源** | `[Snn]`, 发布日期 |

### Step 3：跨事件模式聚合

收集 5+ 个关键事件后，寻找**重复出现的行为模式**：
- 是否在所有危机中都倾向「强硬回击」？
- 是否在所有承诺违背时都倾向「重新解释历史」？
- 是否在所有决策点都优先某种价值（如规模 > 安全）？

## 关键行为分类（聚合后）

| 模式 | 解读 |
|------|------|
| 危机中主动揽责 | 高内控 + 高尽责 + 可能的救世者情结 |
| 危机中转移焦点 | 防御性归因 + 可能的边界调节障碍 |
| 承诺兑现率高 | 高尽责 + 稳定价值观 |
| 承诺事后重释 | 认知灵活性高（正面）/ 自我一致性低（负面） |
| 两难中选择规模 | 增长优先于伦理 |
| 两难中选择保护 | 红线约束思维 + 可能的焦虑型依恋 |

## 最小引证门槛

- 每个关键事件：≥1 条原文描述（来自现场报道 / 声明 / 事后回顾）
- 聚合成模式需要：≥3 个同类事件

## 输出结构

```markdown
### 关键事件分析（CIT）

#### 事件 1：[一句话标题]（YYYY-MM-DD）

- **情境：** [背景]
- **张力：** [面临的选择]
- **实际行为：** 「…」[S04]
- **后果：** [结果]
- **事后解释：** 「…」[S07]（对比实际行为，一致 / 不一致）
- **信号提取：** 此事件揭示 [LoC 外控 / 高神经质 / ...]

（重复 3-6 个事件）

#### 跨事件模式总结

- **模式 1：** [如"危机中倾向于强硬回击"]，出现于事件 1, 3, 5
- **模式 2：** [如"承诺违背后总能重新构造一套解释"]，出现于事件 2, 4

**综合行为预测：** 在[新情境]下，此人极可能采取[行为]，因为历史上在类似情境中[重复模式]。
```
