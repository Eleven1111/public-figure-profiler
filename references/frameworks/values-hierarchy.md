# 价值观层级分析（Values Hierarchy）

> 来源：Shalom Schwartz (1992, 2012), Theory of Basic Human Values（十价值环模型）。
> core.md 3.3 的"价值词频"只能测出此人**说**自己重视什么。
> 本框架测的是价值的**层级**：当两个价值被迫二选一时，哪个赢？
> 真实的价值排序只在**取舍时刻**暴露。

## 第一步：价值识别（Schwartz 十价值映射）

把语料中的价值表达映射到 Schwartz 十价值，统计出现密度与语境：

| 价值 | 核心关切 | 典型表达 |
|------|---------|---------|
| 自主（Self-Direction） | 独立思考与行动 | 「自己判断」「不被定义」「独立思考」 |
| 刺激（Stimulation） | 新奇与挑战 | 「无聊是最大的敌人」「下一个边疆」 |
| 享乐（Hedonism） | 愉悦 | 「享受过程」「快乐最重要」 |
| 成就（Achievement） | 按社会标准的成功 | 「赢」「第一」「证明自己」 |
| 权力（Power） | 地位、支配、资源控制 | 「掌控」「话语权」「地盘」 |
| 安全（Security） | 稳定、秩序 | 「活下去」「护城河」「风险可控」 |
| 遵从（Conformity） | 不越界、克制 | 「守规矩」「不出格」 |
| 传统（Tradition） | 文化/信仰承诺 | 「祖训」「传承」「敬畏」 |
| 仁慈（Benevolence） | 照顾身边人 | 「对团队负责」「不辜负」 |
| 普世（Universalism） | 全人类福祉、公平 | 「社会价值」「普通人」「公平」 |

**结构提示：** 十价值在环上相邻者兼容、相对者冲突（如权力↔普世、自主↔遵从）。
若某人声称同时高度重视环上相对的两个价值，这本身就是一条待解释的矛盾，应进矛盾日志。

## 第二步：取舍时刻定位（核心操作）

在语料（含行为时间线）中定位**价值被迫二选一的时刻**：

- 裁员决策：仁慈 vs 安全/成就
- 增长 vs 合规：成就/权力 vs 遵从/安全
- 套现 vs 控制权：享乐/安全 vs 权力/自主
- 公开表态 vs 沉默自保：普世/自主 vs 安全
- 家族/老臣 vs 专业化：仁慈/传统 vs 成就

对每个取舍时刻记录：情境、两侧价值、**实际选择**（优先用行为证据而非本人解释）、
本人的事后叙事（叙事与选择的落差本身是证据）。

## 第三步：层级重建

1. 按取舍结果排出**显示性价值排序**（revealed hierarchy）
2. 按词频与自述排出**宣称性价值排序**（stated hierarchy）
3. 对比两表：**一致项 = 真实核心价值；落差项 = 表演性价值或自我认知盲区**，落差进矛盾日志

## 最小引证门槛

- 至少 2 个有行为证据的取舍时刻；不足时只输出宣称性排序并明确标注「未经取舍验证」
- 每个进入层级前 3 位的价值：≥2 条引证

## 输出结构

```markdown
### 价值观层级（Values Hierarchy）

**宣称性排序（他说的）：** 1. … 2. … 3. …
**显示性排序（他做的）：** 1. … 2. … 3. …

**取舍时刻分析：**
| 时刻 | 对决价值 | 实际选择 | 证据 |
|------|---------|---------|------|
| [事件] | A vs B | A | 「…」[Snn] + [行为记录] |

**宣称 vs 显示落差：** [落差项及其解释假设]

**核心价值判定：** [1-3 个经取舍验证的价值]，置信度 [高/中/低]

**行为预测（情境 + 行为 + 时限）：**
- 当[价值 X 与价值 Y 再次冲突的具体情境]时，此人会[选择]，因为显示性层级中 X > Y。
```
