# 暗黑三联征（Dark Triad）

> 来源：Paulhus & Williams (2002), "The Dark Triad of personality: Narcissism, Machiavellianism, and psychopathy".
>
> **⚠️ 本框架默认关闭，需显式 `--frameworks dark-triad` 激活。**

## 伦理边界（强制遵守）

1. **不做临床诊断：** 输出必须使用「倾向」「特征」，禁用「患有」「反社会人格障碍」等诊断术语
2. **不针对未同意公开讨论此类特征的人物：** 对**未公开政治角色、未涉及公共利益纠纷**的人物，不激活此框架
3. **高引证门槛（≥5 条跨情境）：** 低于此门槛一律标 `N/A`
4. **配对基准：** 激活此框架时**必须同时跑 Big Five**，交叉验证（避免孤立标签化）
5. **对冲表达：** 所有结论用「在公开表达中呈现出 X 倾向的若干特征」而非「他是 X」

---

## 三大子维度

### N — Narcissism（自恋倾向）

**操作化定义：** 夸大自我重要性、需要持续赞美、缺乏对他人的同理心。

**公开表达指标：**

#### 显性自恋（Grandiose Narcissism）
- 大量第一人称夸赞性表达：「最好的」「无人能及」「我改变了」
- 对成就的反复强调；低估同侪
- 对批评反应激烈；反击优于反思
- 频繁对比性表达：「不像他们那样…」「我的不同在于…」

#### 隐性自恋（Vulnerable Narcissism）
- 表面谦虚，实则敏感
- 对未被足够认可的怨念（"没人看到我做的X"）
- 轻微批评触发长篇辩护

**标注：** `显性自恋倾向` / `隐性自恋倾向` / `混合` / `低` / `N/A`

### M — Machiavellianism（马基雅维利倾向）

**操作化定义：** 对他人的工具化、操纵倾向、道德灵活性。

**公开表达指标：**
- 把关系描述为"有用/没用"
- 对"如何影响他人"有细致的操作性描述
- 将道德语言工具化（用道德话术达到非道德目的的表达）
- **言行落差**：公开倡导的价值与实际行为系统性不一致
- 对"规则"的灵活态度：「原则上…但实际上…」

**关键诊断点：** 马基雅维利不是"没底线"，而是**把道德视为可计算变量**。公开表达中会表现为**对道德成本的精算语言**。

**标注：** `高` / `中` / `低` / `N/A`

### P — Psychopathy（精神病态倾向）

**操作化定义：** 冲动、缺乏同理心、低悔恨感、反社会倾向。

**⚠️ 在公开人物分析中这是最敏感的维度，适用极少。** 大多数功能性高、有长期建设性成就的人物不会显示此倾向。

**可能的公开表达指标：**
- **情感浅层性：** 对他人痛苦/损失的反应单薄甚至不一致
- **冲动性：** 无预警的重大决策；事后无反思
- **寄生性 / 剥削性叙事：** 把他人成就归功于自己
- **缺乏真实悔恨：** 道歉时的空洞化（无具体承担、无改变承诺）
- **魅力表面下的冷漠：** 口才流利但内容空洞

**标注：** `可能的 P 倾向信号（需高度谨慎）` / `未观察到` / `N/A`

## 交叉验证规则

### 与 Big Five 的一致性检查

| 暗黑维度 | 对应的 Big Five 低分 | 若不一致则降低置信度 |
|---------|---------------------|---------------------|
| Narcissism | 低 Agreeableness | 若 A 高但 N 显性，需重新审视 |
| Machiavellianism | 低 Agreeableness + 低 Conscientiousness（道德维度） | |
| Psychopathy | 低 Agreeableness + 高 Neuroticism（冲动一面） + 低 Conscientiousness | |

### 与 EMS 的交叉参考

- 显性自恋常伴随 **特权/高人一等图式**
- 隐性自恋常伴随 **缺陷/羞耻 + 寻求认可图式**
- 马基雅维利常伴随 **不信任/虐待图式 + 情感抑制图式**

---

## 最小引证门槛

- 每个子维度：**≥5 条跨情境引证**
- 至少覆盖 **2 种不同受众场合**
- 至少 **1 条非当事人自述**（第三方报道 / 访谈他人 / 意外泄露）
- 不满足此门槛一律标 `N/A`
